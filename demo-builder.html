How to put a generated demo online (free)

Keep one GitHub repo named demos with Pages enabled. For each business, add a folder with its file: demos/apex-plumbing/index.html. Its link becomes yourname.github.io/demos/apex-plumbing/. All demos share one origin, so one repo + one Pages setup covers every client.

Lock the worker / change the NVIDIA key

Lock to your domain: in the worker code set const ALLOWED_ORIGINS = ["https://yourname.github.io"]; then Deploy. (Leave empty while testing.)

Change/rotate the key: Cloudflare → your worker → Settings → Variables and Secrets → edit NVIDIA_API_KEY → paste new key → Deploy.
